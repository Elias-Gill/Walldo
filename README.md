# Walldo, a simple wallpaper changer for windows.
## Why ?...
Because I'm the type of person who likes to change his wallpaper all the time (and dislike the idea of automatic
changes). There are a lot of apps like this around, but they are bloat (with crazy customs and things no one ever uses)
and they aren't fast and simple to use. That's why I decide to create this app for my self.

## Electron, that's awfull man...
Yes, i know, electron is well know to be the opositte of blasingly fast, but I'm a idle person, so, i don't wanna spend
time (wich i don't have) trying to make something with some magic language with his horrible libraries. With
HTML, CSS and JS, is really easy to make UI's.
